Профессия Front-End-разработчика

Front-End - в наше время, пожалуй, быстрее всего развивающаяся область програмиррования.С другой стороны, эти технологии достаточно устоявшиися и распространены практически повсеместо, ведь браузер есть во многих электронных устройствах.

Курс
Front-End Basic

Front-end basic

В рамках курса Front-end basic студенты без опыта в разработке изучают HTML и CSS. Нарабатывают практические навыки в верстке адаптивных сайтов, знакомятся с пре- и постпроцессорами. В программу курса также входит CSS фреймворки, такие как Bootstrap. По окончанию курса успешние Выпускники могут занимать должность верстальщика в Web-стдиях.

advanced

Front-end pro

Курсы Front-End Pro предназначены для Студентов, имеющих практические навыки в верстке. По окончании курса Выпускники владеют JavaScript на достаточном уровне, что бы писать скрипты, формы, работать с AJAX, а так же могут настроить сборку проекта в Gulp. Курс позволяет претендовать на вакансии Juniour Front-End Developer.
